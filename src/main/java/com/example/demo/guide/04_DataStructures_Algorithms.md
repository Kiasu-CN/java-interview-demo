## 面试 DP 通用解题步骤

1. **定义状态**：`dp[i]` 或 `dp[i][j]` 代表什么？
2. **推导状态转移方程**：`dp[i] = ?`（最关键的一步）
3. **确定边界条件**：`dp[0] = ?`
4. **确定遍历顺序**：从左到右？从上到下？
5. **空间优化**：能否压缩为一维数组？

## 面试回溯通用模板

```
void backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择  ← 这就是"回溯"
```
