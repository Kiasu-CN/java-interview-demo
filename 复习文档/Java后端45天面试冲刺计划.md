# Java后端45天面试冲刺计划（精简执行版）

目标：45 天内形成可面试输出能力（算法 + 八股 + 项目表达）。
配套文档：
1. `Java后端面试分层题库.md`（题库）
2. `LeetCode必刷题单_Hot100优先.md`（算法）

---

## 每日固定节奏（6h+）

1. 算法（2h）：2题（至少1题 medium）
2. JavaGuide专题阅读（2.5h）
3. 口述题（1h）：12题（8必会 + 3追问 + 1加分）
4. 复盘（0.5h）：错题与次日计划

## 验收标准（每日）

1. 算法：2题完整提交 + 1条错因
2. 八股：12题口述

## AI使用规则

1. 每题先独立思考/编码 30 分钟
2. AI只用于查错、补追问、复盘
3. 每天至少 1 次脱稿口述（5-10分钟）
