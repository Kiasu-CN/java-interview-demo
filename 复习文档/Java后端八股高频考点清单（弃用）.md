# Java 后端八股全量复习清单（可从初面到深挖）

使用方式：
1. 先刷“一级必会（初面高频）”。
2. 再刷“二级进阶（复试追问）”。
3. 最后刷“三级场景（项目结合）”。

口述模板（每题 60-90 秒）：结论 -> 原理 -> 场景 -> 风险/边界。

---

## 0. Java 基础与语言机制

### 一级必会
1. `==` 与 `equals` 区别。
2. `hashCode` 作用，为什么重写 `equals` 必须重写 `hashCode`。
3. 基本类型与包装类型区别，自动装箱/拆箱风险。
4. `String`、`StringBuilder`、`StringBuffer` 区别。
5. `final`、`finally`、`finalize` 区别。
6. `try-catch-finally` 执行顺序与返回值覆盖。
7. 重载与重写区别。
8. 接口与抽象类区别。

### 二级进阶
1. 泛型擦除原理，桥方法是什么。
2. 反射机制与性能影响。
3. 注解保留策略（SOURCE/CLASS/RUNTIME）。
4. SPI 机制与典型场景（JDBC 驱动）。
5. 序列化与反序列化风险。

### 三级场景
1. 线上 NPE 如何定位（日志、参数、链路）。
2. 如何设计可维护的异常体系（业务异常/系统异常）。

---

## 1. 集合框架（重点）

### 一级必会
1. `ArrayList` 与 `LinkedList` 区别（结构、复杂度、场景）。
2. `ArrayList` 扩容机制。
3. `HashMap` 底层结构（JDK8：数组+链表+红黑树）。
4. `HashMap` 的 `put/get` 流程。
5. 为什么 `HashMap` 容量通常为 2 的幂。
6. `HashMap` 冲突处理与树化条件。
7. `HashSet` 去重原理。
8. `TreeMap` 与 `LinkedHashMap` 典型场景。
9. `fail-fast` 是什么，为什么会 `ConcurrentModificationException`。

### 二级进阶
1. JDK7 与 JDK8 `HashMap` 差异。
2. `LinkedHashMap` 如何实现 LRU 思路。
3. `Comparable` 与 `Comparator` 区别。
4. `CopyOnWriteArrayList` 原理与适用场景。

### 三级场景
1. 百万级去重为何不直接 `HashSet`（内存与顺序问题）。
2. 读多写少场景为什么考虑 `CopyOnWriteArrayList`。

---

## 2. 并发编程（重点）

### 一级必会
1. 线程创建方式：`Thread` / `Runnable` / `Callable`。
2. 线程池核心参数与拒绝策略。
3. 为什么不推荐 `Executors` 默认工厂。
4. `synchronized` 与 `ReentrantLock` 区别。
5. `volatile` 语义（可见性、有序性，不保证原子性）。
6. `CAS` 原理与 ABA 问题。
7. `AQS` 核心思想。
8. `ConcurrentHashMap` 为什么线程安全。
9. `ThreadLocal` 原理与内存泄漏风险。
10. `CountDownLatch` / `CyclicBarrier` / `Semaphore` 区别。

### 二级进阶
1. 锁优化：偏向锁、轻量级锁、重量级锁（JDK 版本变化可简述）。
2. 线程池参数如何根据 CPU 密集/IO 密集估算。
3. 并发死锁如何排查（`jstack`）。
4. `StampedLock` 与读写锁差异。

### 三级场景
1. 接口限流如何设计（令牌桶/漏桶）。
2. 高并发下超卖如何避免（锁+幂等+库存扣减策略）。

---

## 3. JVM（重点）

### 一级必会
1. 运行时数据区：堆、栈、方法区、程序计数器、本地方法栈。
2. 对象创建流程与对象头概念。
3. GC 可达性分析。
4. 常见 GC 算法（标记清除、复制、标记整理）。
5. 常见回收器（G1/CMS 基本特点）。
6. 类加载过程（加载/验证/准备/解析/初始化）。
7. 双亲委派模型与价值。
8. 常见 OOM 类型。

### 二级进阶
1. GC 日志怎么看（暂停时间、频率）。
2. 常见 JVM 参数（堆大小、新生代比例、GC 选择）。
3. Full GC 频繁排查思路。

### 三级场景
1. 线上内存飙升定位路径（监控 -> dump -> MAT 分析）。

---

## 4. Spring / Spring Boot / Spring MVC

### 一级必会
1. IOC 与 DI。
2. Bean 生命周期。
3. AOP 原理（JDK 动态代理 vs CGLIB）。
4. 事务传播行为与隔离级别。
5. 事务失效常见场景（同类自调用、异常处理不当）。
6. Spring Boot 自动配置原理。
7. `@SpringBootApplication` 组成。
8. Spring MVC 请求处理流程。
9. 拦截器、过滤器、AOP 区别。
10. 参数校验与全局异常处理。

### 二级进阶
1. 循环依赖如何解决。
2. BeanFactory 与 ApplicationContext 区别。
3. 条件装配与自定义 starter 思路。

### 三级场景
1. 如何设计统一返回体与错误码。
2. 如何设计幂等中间件（token/唯一键/状态机）。

---

## 5. MyBatis

### 一级必会
1. `#{}` 与 `${}` 区别。
2. MyBatis 执行流程。
3. 一级缓存与二级缓存区别。
4. 动态 SQL 常见写法。

### 二级进阶
1. 插件机制（拦截器）原理。
2. 分页实现方式与插件原理。

### 三级场景
1. SQL 注入风险如何规避。

---

## 6. MySQL（重点）

### 一级必会
1. 为什么是 B+ 树索引。
2. 聚簇索引/二级索引、回表、覆盖索引。
3. 联合索引最左前缀。
4. 索引失效常见情况。
5. ACID 与隔离级别。
6. MVCC、快照读、当前读。
7. 行锁、间隙锁、Next-Key Lock。
8. Explain 核心字段（type、key、rows、extra）。
9. 慢 SQL 优化基本路径。

### 二级进阶
1. 深分页优化。
2. 死锁案例分析与规避策略。
3. 主从复制基本原理。

### 三级场景
1. 高并发写入下如何做分库分表决策。

---

## 7. Redis（重点）

### 一级必会
1. 数据结构及场景（String、Hash、List、Set、ZSet）。
2. 缓存穿透/击穿/雪崩及解决方案。
3. 过期策略与淘汰策略。
4. RDB 与 AOF 区别。
5. 分布式锁实现要点。

### 二级进阶
1. 锁续期、防误删、可重入设计。
2. 主从、哨兵、Cluster 区别。
3. 大 Key/热 Key 问题治理。

### 三级场景
1. 缓存一致性方案（延时双删、旁路缓存、异步补偿）。

## 8. 计算机网络与安全

### 一级必会
1. TCP 三次握手、四次挥手。
2. HTTP 与 HTTPS 区别。
3. Cookie / Session / JWT 区别。
4. GET 与 POST 区别。
5. 常见状态码。
6. XSS / CSRF / SQL 注入。

### 二级进阶
1. HTTPS 握手核心流程（证书、密钥协商）。
2. 长连接、短连接与 Keep-Alive。

## 9. 设计模式与工程实践

### 一级必会

2. 工厂、策略、模板方法在业务中的落地。

### 二级进阶
1. 观察者、代理模式实战场景。

## 12. 复习排期建议（6h+/天）

1. 第 1 周：Java 基础+集合+并发。
2. 第 2 周：JVM+Spring+MyBatis。
3. 第 3 周：MySQL+Redis+网络安全。
4. 第 4 周：项目场景题+模拟面试。
5. 第 5-6 周：查漏补缺+二轮强化。

## 13. 自测标准（是否“真正掌握”）

1. 每题 90 秒内可口述主干。
2. 每个模块至少能回答 3 个追问。
3. 每个知识点能关联一个项目场景。
4. 不看资料可写出关键代码骨架（如 LRU、线程池配置、缓存旁路）。
