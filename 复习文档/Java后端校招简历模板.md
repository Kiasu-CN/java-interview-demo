## Java后端岗位加分写法

- 项目描述尽量写清楚：**业务场景 + 技术栈 + 你的职责**
- 项目亮点尽量写成：**做了什么 + 用了什么技术 + 带来了什么效果**
- 如果你写了 Redis，最好能讲清缓存场景、常见问题和解决思路
- 如果你写了 MySQL，最好能讲清索引、事务、慢 SQL 优化
- 如果你写了并发或秒杀，最好能讲清超卖、幂等、削峰、限流这些关键词
- 简历上写到的每个技术点，都要准备好面试时展开说明
